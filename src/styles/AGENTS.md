# Styles Directory

## Purpose
Global styling and design system configuration.

## Files

### `globals.css`
**Source:** Design system adapted from cogni-site  
**Features:**
- Tailwind CSS imports (`@tailwind base;components;utilities;`)
- CSS variables for dark/light theme colors  
- Inter and Merriweather font imports from Google Fonts
- Dark theme as default with semantic color variables  
- Component-specific classes for interactive elements

**Color System:**
- Primary: Blue (`--primary`, `--primary-foreground`)
- Secondary: Dark gray (`--secondary`, `--secondary-foreground`)  
- Background: Black (`--background`, `--foreground`)
- Accent: Electric blue (`--accent`, `--accent-foreground`)
- Borders, inputs, and other UI elements with consistent theming

**Typography:** Inter (sans-serif) and Merriweather (serif) loaded from Google Fonts
